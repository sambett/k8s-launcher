⚠️ Read before you start — Platform requirements
Supported OS Ubuntu 22.04 LTS or Ubuntu 24.04 LTS — all nodes (control plane + workers) must run the same version.
Architecture amd64 (x86-64) only.
Why same OS? Longhorn relies on kernel modules — iSCSI, device-mapper — that differ between Ubuntu LTS generations. A mixed-OS cluster produces subtle, hard-to-diagnose storage failures under load.
