GPU Node Prerequisites

ℹ Before running this check — the following steps must be completed manually on the GPU node before this platform can configure it:
  1. Install NVIDIA driver — sudo apt install nvidia-driver-<version> && sudo reboot then verify with nvidia-smi
  2. Install NVIDIA Container Toolkit — sudo apt install nvidia-container-toolkit  (requires the NVIDIA apt repo)
This platform automates only step 3 (containerd runtime config) and step 4 is verified automatically after restart. All four checks must be green before installing the Device Plugin.

Driver and toolkit must be installed manually on the node. The containerd runtime configuration is the only step this platform fixes automatically.
