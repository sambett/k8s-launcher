GPU Support — NVIDIA Device Plugin checking...

The NVIDIA Device Plugin runs as a DaemonSet on every node. It communicates with the kubelet to advertise nvidia.com/gpu as a schedulable Kubernetes resource. Without it, GPU pods stay Pending regardless of what drivers are installed.

Prerequisites per GPU node (done outside this launcher):
1. Install NVIDIA drivers on the VM (apt / runfile)
2. Install NVIDIA Container Toolkit on the VM
3. Configure containerd to use the NVIDIA runtime
Once those are done, label the node below and it will automatically advertise its GPU count.

Label a node as GPU-capable

Labels the node accelerator=nvidia. This is what tells the device plugin and your JupyterHub GPU profiles which nodes have GPUs. Only label nodes that physically have an NVIDIA GPU and the required drivers installed.
