🦊 Reset GitLab

Uninstalls GitLab CE completely from the VM — stops all services, purges the package, and deletes all data under /etc/gitlab, /var/opt/gitlab, and /opt/gitlab. Also clears all generated credential files on the controller. Re-run the GitLab tab to start fresh.

📓 Reset JupyterHub

Uninstalls the JupyterHub Helm release and deletes the jhub namespace including all secrets and ConfigMaps. Clears the generated JupyterHub vars file. Re-run the JupyterHub tab to redeploy.

Note: Longhorn volumes use a Retain reclaim policy — old user data volumes remain in Longhorn until manually deleted.

⚙ Reset Admin Dashboard

Stops and removes the workbench-admin systemd service and deletes /etc/workbench/ (secrets). Clears the generated dashboard vars file. Re-run the Admin Dashboard tab to redeploy.

Cluster Reset

Tears down the Kubernetes cluster and/or all generated state. None of these actions can be undone.
