Add worker node

Joins a new VM to the running cluster. The password is used once to push the SSH key and is never stored. The node will be configured identically to the existing workers (~10 minutes).

🖥️
Node OS requirement

The new worker must run Ubuntu 22.04 LTS or Ubuntu 24.04 LTS, and must match the OS version of the existing workers.

This platform pins every component — Kubernetes, containerd, Calico, Longhorn — to a validated, tested combination. Longhorn in particular depends on kernel modules (iSCSI, device-mapper) whose behaviour differs between kernel generations. A mixed-OS cluster can produce subtle storage failures that are very hard to diagnose.

This is standard practice: AWS EKS, Google GKE, and NVIDIA AI Workbench all enforce OS version constraints on their nodes for the same reason.
